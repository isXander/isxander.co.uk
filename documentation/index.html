Xander Client is a project maintained by isXander.
It is a project dedicated to bringing the best, lag-free, experience to hypixel players.
Many features are planned that no other clients offer.

Where can I get this?
This client is currently in early alpha stages and has not yet been released to the public.

What can I do to help the project?
You could join the discord server or maybe even apply to become a developer!
